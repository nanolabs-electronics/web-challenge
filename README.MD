Prerequisites:

    JavaScript knowledge

Instruction:

    clone or download the contents of this repository
    solve the challenges
    create a .zip archive of your code
    send an email at office@nanolabs.ro with the .zip archive as an attachment

The purpose of this challenge is to check both the candidate's JavaScript knowledge as well as basic algorithmic skills. Inside this repository you will find a problem statement.  
The problem can be solved however you wish, with the only requirement being that the solution MUST be written using JavaScript.  
There is no time limit for solving the challenge, but a timely delivery is appreciated.  

We recommend solving these challenges on your own, with minimal external help (people). Still, you are free to use internet resources when stuck.
